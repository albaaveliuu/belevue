## Installation

1. Stelle sicher, dass Node.js installiert ist
2. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
3. Starte die Entwicklungsumgebung:
   ```bash
   npm start
   ```
4. Öffne [http://localhost:3000](http://localhost:3000) im Browser

## Build für Produktion

```bash
npm run build
```
